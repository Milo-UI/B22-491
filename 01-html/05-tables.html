Tables

Struttura base

  • <table> -> contenitore della tabella
  • <tr> -> table row: una riga
  • <th> -> table header: cella di intestazione (testo in grassetto e centrato di default)
  • <td> -> table data: cella normale

Attributi utili su <th> e <td>

  • colspan="2" -> la cella occupa 2 colonne
  • rowspan="3" -> la cella occupa 3 righe

Elementi opzionali (buona pratica semantica)

  • <thead> -> raggruppa le righe di intestazione
  • <tbody> -> raggruppa le righe di dati
  • <tfoot> -> raggruppa le righe di piè di tabella

Non cambiano la resa visiva di default, ma migliorano la leggibilità del codice e l'accessibilità.
